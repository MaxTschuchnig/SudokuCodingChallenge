## 🚀 Running the Project Locally

To run the project locally, ensure the following services are running:  

- ✅ **MongoDB** must be running  
- ✅ **Compiled Sudoku Generator** must be running (or enough sudokus must be pre-generated)  
- ✅ **Webserver** must be running and pointing to the Sudoku Generator `/data` folder  

---

## 🌍 Running on a Server

For a server deployment, you additionally need:  

- 🌐 A **public IP address**  
- 🔄 **Nginx** as a reverse proxy (as configured in Nginx settings)  
- 🏡 *(Optional)* A **landing page**  
- 🔒 *(Optional)* **HTTPS** for security  
- 🌍 *(Optional)* A **DNS name** for easy access  

Happy hosting! 🚀
